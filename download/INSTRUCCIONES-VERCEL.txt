¡Hola! Aquí tienes tu aplicación Goopi lista para subir a Vercel.

═══════════════════════════════════════════════════════════════
📋 PASO 1: CREAR CUENTA EN VERCEL (GRATIS)
═══════════════════════════════════════════════════════════════

1. Ve a: https://vercel.com
2. Haz clic en "Sign Up" (Registrarse)
3. Regístrate con tu cuenta de Google o GitHub (es más fácil)

═══════════════════════════════════════════════════════════════
📋 PASO 2: CREAR CUENTA EN GITHUB (GRATIS)
═══════════════════════════════════════════════════════════════

1. Ve a: https://github.com
2. Haz clic en "Sign Up" (Registrarse)
3. Crea tu cuenta gratis

═══════════════════════════════════════════════════════════════
📋 PASO 3: SUBIR TU PROYECTO A GITHUB
═══════════════════════════════════════════════════════════════

1. En GitHub, haz clic en "New repository" (Nuevo repositorio)
2. Ponle nombre: "goopi-app"
3. Haz clic en "Create repository"
4. Luego sube los archivos del ZIP:
   - Descomprime el archivo "goopi-app.zip"
   - Sube todos los archivos a tu repositorio

═══════════════════════════════════════════════════════════════
📋 PASO 4: CONECTAR CON VERCEL
═══════════════════════════════════════════════════════════════

1. En Vercel, haz clic en "Add New" → "Project"
2. Selecciona "Import Git Repository"
3. Elige tu repositorio "goopi-app"
4. Haz clic en "Import"
5. Haz clic en "Deploy"
6. ¡Listo! Tu app estará online en segundos

═══════════════════════════════════════════════════════════════
⚠️ IMPORTANTE: CONFIGURAR FIREBASE
═══════════════════════════════════════════════════════════════

Para que funcione el login, necesitas la API Key de Firebase:

1. Ve a: https://console.firebase.google.com
2. Selecciona tu proyecto: "taxi-macas-52717"
3. Ve a configuración (icono de engranaje)
4. Copia la "apiKey" 
5. En Vercel, ve a tu proyecto → Settings → Environment Variables
6. Agrega esta variable:
   - Nombre: NEXT_PUBLIC_FIREBASE_API_KEY
   - Valor: [tu API key de Firebase]
